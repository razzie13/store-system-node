RETAIL STORE SYSTEM PROJECT

by nessumsar

uploaded to public_html ONLY

This project emulates the back end of a small retail shop and allows staff to manage the store e-commerce page, create store signage, and use the store's point-of-sale system, as well as maintain information on each of the store's available items.

    The following are the available components of the project:
  • Store E-Commerce Site
  • Store Point-Of-Sale System
  • Store Sign-Making Kit (Full Page Signs)
  • Easy-to-print cashier produce code reference page
    Significant parts of this project include:
  • Creating a detailed and easy-to-use product database and schema for all the characteristics of each item
  • Creating a point-of-sale setup which automatically reads if an item has promotional pricing, adding applicable taxes, asking for an item weight if the item is sold by weight, and allowing the system to void any item a customer no longer wants
  • Creating an e-commerce web page that lists the store's available items automatically using the database information
  • Creating a way to easily make in-store signage which automatically uses promo pricing if an item is on sale, converts kilogram prices to pound prices, and prompts for Country of Origin information if a produce sign is being made
